Heat¶

  • Build and test history
  • Licensed under BSD-2

Solve the heat equation with constant coefficients, heat source, and usual boundary conditions using Green’s function on a line (1D), a rectangle (2D), or a block (3D). Documentation is available here

Requirements¶

  • Unix system with Python 2.7/3.5
  • C++ compiler supporting the C++11 standard
  • cmake

Download and test¶

  1. git clone https://github.com/frRoy/Heat.git
  2. cd Heat
  3. pip install –upgrade wheel setuptools pip
  4. pip install -r requirements/development.txt
  5. mkdir build && cd build
  6. pip install -e .
  7. cmake ..
  8. make
  9. ctest
